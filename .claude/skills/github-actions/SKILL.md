# GitHub Actions

## Principles

- mirror local scripts
- fail fast on lint and type errors
- parallelize independent jobs
- cache package-manager artifacts safely
- keep workflow permissions minimal

## Apply When

- creating or tuning CI pipelines
- adding preview, release, or deploy workflows
- speeding up slow builds and test suites
