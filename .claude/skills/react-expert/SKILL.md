# React Expert

## Principles

- keep rendering code simple and compositional
- extract hooks only when behavior repeats or needs testable separation
- treat forms, async states, and accessibility as first-class concerns
- optimize only after a real bottleneck is identified

## Apply When

- building components and layouts
- designing hook APIs
- fixing rendering bugs or hydration issues
- improving UX states: loading, error, empty, optimistic
