# Vitest Testing

## Principles

- keep tests fast and deterministic
- prefer user-visible behavior over implementation detail assertions
- use factories or builders for readable setup
- run targeted tests during iteration and the broader suite before completion

## Apply When

- testing React components
- testing utility modules and client-side state
- writing fast integration coverage in Vite-based repos
