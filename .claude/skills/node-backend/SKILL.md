# Node Backend

## Principles

- keep process startup deterministic
- handle retries, timeouts, and cancellation deliberately
- treat env parsing and secret access as boundary code
- keep side effects isolated behind focused modules

## Apply When

- writing workers, jobs, cron tasks, or scripts
- integrating with external APIs or queues
- building shared backend utilities outside a framework
