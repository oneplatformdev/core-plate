# Debugging Wizard

## Workflow

1. reproduce
2. isolate
3. instrument
4. confirm root cause
5. fix minimally
6. add regression coverage

## Apply When

- behavior differs between environments
- issues are flaky or timing-related
- hydration, caching, async, or DI problems appear
