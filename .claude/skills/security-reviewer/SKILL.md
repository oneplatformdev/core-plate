# Security Reviewer

## Checklist

- auth and authorization boundaries
- unsafe input paths
- secret leakage
- insecure token storage
- SSR/client bundle data exposure
- upload and file-serving risks
- CORS, CSRF, and webhook signature handling

## Apply When

- touching auth or access control
- exposing new endpoints
- integrating third-party systems
- handling files, sessions, or privileged server logic
