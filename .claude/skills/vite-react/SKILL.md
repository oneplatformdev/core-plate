# Vite React

## Principles

- keep app bootstrap simple
- centralize environment-variable access
- separate route/page composition from shared UI primitives
- use fast feedback tools: Vitest, ESLint, and build checks

## Apply When

- building SPA features
- configuring Vite entry points, aliases, and env access
- shaping client-only data flows and API clients
