# Playwright Expert

## Principles

- automate high-value user journeys, not every click
- prefer resilient selectors tied to roles or labels
- keep tests isolated and environment-aware
- capture enough trace output to debug failures

## Apply When

- writing end-to-end flows
- reproducing browser regressions
- verifying auth, navigation, or checkout-like critical paths
